# Jenkins

## 安装

#### 版本

最新版需要JDK11、17的支持
我不想再装JDK所以使用的是历史版本 - [2.346.1 下载链接](https://get.jenkins.io/war-stable/2.346.1/)    [历史版本 列表](https://get.jenkins.io/war-stable/)

启动命令

~~~bash
java -jar jenkins.war
~~~

默认密码 C:\Users\sun\.jenkins\secrets\initialAdminPassword
